# Java Calculator Application

A menu-driven calculator application developed in Java using Object-Oriented Programming principles.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Square Root
- Power Calculation
- Sine Function
- Cosine Function
- Tangent Function
- Logarithm (Base 10)
- Natural Logarithm (ln)
- Input Validation
- Calculation History
- Save History to File
- Clear History
- Exception Handling

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (ArrayList)
- File Handling
- Git
- GitHub
- VS Code

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Compile the Java files.
4. Run Main.java.
