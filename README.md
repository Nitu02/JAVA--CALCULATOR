# Java Calculator Application

A menu-driven calculator built using Java.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Square Root
- Power Function
- Calculator History
- Input Validation
- Exception Handling

## Technologies Used

- Java
- VS Code
- Git
- GitHub

## How to Run

1. Clone the repository
2. Open in VS Code
3. Compile:

```bash
javac src/*.java
```

4. Run:

```bash
java src/Main
```
